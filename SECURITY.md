# Security Policy

LICTOR is a local-first defensive triage engine.

## Scope

LICTOR is intended for authorized defensive review of indicators, text, emails and local files.

## Rejected behavior

LICTOR must not be modified to perform:

- credential theft;
- spyware behavior;
- hidden sending;
- covert upload;
- unauthorized access;
- destructive automation;
- offensive persistence.
